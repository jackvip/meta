# 紫微斗数
## 游戏视角
### 人生RPG
今天，我将从游戏设计的视角来解读紫微斗数。选择这个视角，主要有两个原因。

第一，我本身是学计算机、做系统构架的，习惯用逆向思维去分析系统设计。其实人生完全可以用一个RPG游戏来模拟。你可能玩过《黑神话：悟空》这类游戏，玩家扮演天命人打怪过关、完成使命，这就是典型的RPG。人生与这种游戏模式非常相似。

第二，我在自学紫微斗数时，发现它的知识结构比较碎片化，各种星曜如果没有系统化的逻辑串联，很难理解。我不建议死记硬背，而是希望找到一套合理的逻辑，把紫微斗数的知识串起来。很多初学者一看到那么多星曜就头疼，感觉只能靠硬背，其实我们可以换个角度，从设计者的视角来构建紫微斗数的人生模型。

一个完整的游戏应该包含以下几个系统：角色系统、辅助系统、阻杀系统、关卡系统、规则系统。下面我们一个个来解析。

### 角色系统
一个人在社会中扮演不同的角色：做官、经商、文人、手艺人、普通人……这些社会分工就是一个个角色。斗数的星曜系统也是由角色构成的，它把人类社会分为四个层级：

**第一层：秩序维护层**  
主要负责社会秩序的维护，典型特征是当官。代表星曜：紫微、天府、武曲、天相、廉贞，共五颗星。  
- 紫微为帝  
- 天府为财库之长  
- 武曲为武官之长  
- 天相为文官之长  
- 廉贞为衙门  
> 这里需要说明，紫府武相廉这五颗星居于命宫就代表一定能当官，只是说明可能有贵气而已，最终能否当官还得看三方四正。

**第二层：文明迭代层**  
主要负责人类文明的迭代，典型特征是富有，因为艺术、文学、科学等创新往往需要经济条件支撑。代表星系：天机、太阴、天同、天梁、巨门、太阳。  
- 天机精于计算  
- 太阴精于规划  
- 天同精于和谐  
- 天梁精于钻研  
- 巨门精于传播  
- 太阳精于博爱  
> 这里需要说明，机月同梁巨日这六颗星居于命宫就代表一定能成为商人，只是说明可能有财气而已，最终是否有才还得看三方四正。

**第三层：破旧立新层**  
主要为了打破陈旧、推动革新，典型特征就是变革。代表星系：七杀、破军、贪狼，合称杀破狼。
- 七杀为主帅精于开拓
- 破军为先锋精于革新主耗
- 贪狼为欲望精于交际

**第四层：芸芸众生层**  
没有典型特征，不成格局，也就是普通人。

### 什么是格局？
在了解角色系统后，你可能会听到格局这个词。格局就是一种固定搭配。生活中也有很多固定搭配，比如鸡蛋+木耳+黄瓜炒出来叫木须肉，虽然里面并没有肉；猪肉+胡萝卜+木耳炒出来叫鱼香肉丝，虽然里面并没有鱼。紫微斗数里也有类似的经典星曜组合，古人把它们提取出来，称为格局。有的格局主大富，有的主大贵，有的大富大贵。而芸芸众生格就是指不符合任何经典组合的普通命盘。

举个例子：紫微+天府+天相+左辅右弼+文昌文曲+天魁天钺+台辅封诰+三台八座，在三方四正出现这样的组合，就叫百官朝拱格，从字面就能理解是皇帝般的命格。如果缺少左辅右弼、文昌文曲等双星，或者只有单星，那就破了百官朝拱，变成紫府朝垣格，地位和成就相对降低。如果只有紫微独坐，三方四正又遇到煞星（火星、铃星、擎羊、陀罗、地空、地劫、化忌），那就再次破格，成为孤君在野，可能一事无成，甚至出家为僧。

以上就是斗数的角色系统。有没有感觉到，它和我们社会的层级是不是完全对应？所以，看到紫府武相廉，就联想到当官的特质；看到机月同梁巨日，就联想到富人特性；看到杀破狼，就联想到革新；剩下的不成格局，就是普通人。这样，14颗主星的轮廓是不是清晰多了？14颗主星还存在着双星系统，但是基本都不会偏离单星星性太远，这个我们放到后面再讲。
### 辅助系统
角色系统之外，还需要辅助系统。辅助系统包括：禄存天马、左辅右弼、文昌文曲、天魁天钺这四对吉星，外加化科、化权、化禄。
- 禄存主财禄，代表稳定且持续的财富。
- 天马主奔波、变动，代表通过动来获利。
- 左辅主同辈、下属的助力，有增加、扩大的意味。
- 右弼也主助力，偏向于暗助或来自女性/贵人的帮助。
- 文昌主正统学术、文书契约、声誉。偏向于传统学识、考试、功名。
- 文曲主才艺天赋、口才表达、流行文化。偏向于非正统的才艺。
- 天魁主直接的、明显的贵人提携或机遇。多为男性长辈或有社会地位的人。
- 天钺主间接的、暗中的贵人相助。多为女性长辈或看似不起眼的人。
- 化禄代表多与顺。主财禄、人缘、享福，是一种柔性的助力。
- 化权代表强与控。主权威性、竞争性，增强掌控力、决断力。
- 化科代表名与和。主科甲、学历、化解纷争。

### 阻杀系统
有辅助就有阻杀。阻杀系统的代表星曜包括：火星、铃星、擎羊、陀罗、地空、地劫，这六颗煞星合称六煞，加上化忌星并称七煞。其中火星、铃星、擎羊、陀罗又叫做四煞。
- 火星，主突发性事件、急性问题、来得快、去得也快。
- 铃星，主慢性问题、内心焦虑、暗中麻烦，持久而难以摆脱。
- 擎羊，主利器之伤、正面冲突、快速决裂。
- 陀罗，主慢性折磨、停滞不前、旧事重来 。
- 地空，主理想高远但易落空、精神追求强。代表空亡，是精神上的耗散。
- 地劫，主现实中的损耗、计划受阻、事与愿违。代表劫难，是现实层面的阻碍。
### 关卡系统
关卡系统就是十二宫。按顺时针排列为：命宫、父母宫、福德宫、田宅宫、官禄宫、交友宫、迁移宫、疾厄宫、财帛宫、子女宫、夫妻宫、兄弟宫。
- 命宫：先天性格、外貌气质、整体运势格局，它决定了其他十一宫的基调
- 兄弟：手足、同辈、竞争的关系与缘分，也可看合伙人的初始状态。
- 夫妻：配偶的性格、婚姻质量、感情模式、也反映一个人对感情的态度
- 子女：子女、晚辈、下属的缘分、以及个人的生育能力和桃花
- 财帛：财运、理财、收入。看的是如何得财和现金流，而非总资产
- 疾厄：先天体质、易患疾病、身体薄弱环节，也反映一个人的情绪、意外灾祸的倾向
- 迁移：外出发展的际遇、人际关系中的表现、社会形象、外地发展的运势
- 交友：朋友、同事、合作伙伴的缘份与互动，此宫更偏社会交往与利益关系
- 官禄：事业发展方向、工作能力、管理能力、社会地位、看的是事业形态与成就高度
- 田宅：家庭环境、祖业根基、房产运、居住条件、晚年生活、家族长辈的庇护
- 福德：精神享受、内心世界、福报深浅、看一个人是否快乐、有什么精神追求、晚年的心境。
- 父母：母缘分、遗传基因、长辈助力、教育背景、上司关系、政府机构、文书契约

我们一生遇到的阻力、机遇，都分布在这十二个关卡中。
### 规则系统
规则系统分为宏观规则和微观规则。

**微观规则**  
从游戏设计角度看，这套程序的初衷是通过批量个体的成败来推动整个人类文明的迭代。想象这是一个大型网络游戏，每位玩家上线后，就被分配一套主星+辅星+煞星+关卡的布局。玩家的任务是带领这套配置通关。关卡通过时间轴触发，分为大限（10年）、流年、流月、流日、流时。每10年的任务重心不同。

那么紫微斗数的作用是什么？它相当于一本通关手册，告诉你每一关会发生什么事件，以及事件发生的时间。

**宏观规则**  
宏观规则就是天道、阴阳、五行、因果。有人说《易经》64卦揭示了64种规律，这些规律就是宏观规则。有意思的是，古人说人教人教不会，事教人一次就会，造物主正是让我们在每个关卡中去领悟这些宏观规则。

你可能会问：紫微斗数怎么知道每一关会发生什么、什么时候发生？这正是我们学习紫微斗数的意义——它破解了关卡的组成逻辑和触发时间点。通过紫微斗数的规则，我们可以推演每年发生的事件及其时间。听起来很厉害？不过别着急，学习紫微斗数需要循序渐进，我们接下来会一步步深入它的规则逻辑。

### 总结
我们再通过这张思维导图来回顾一下紫微斗数规则下的这个游戏系统。若你已经理解了这场人生游戏，那么后面的紫微斗数就是在这个大框架下不断细化，让我们一起来探索。

## 基本术语
1. 正曜：指紫微、天机、太阳、武曲、天同、廉贞、天府、太阴、贪狼、巨门、天相、天梁、七杀、破军等十四曜。
2. 辅曜：指左辅、右弼，天魁、天钺四曜。
3. 佐曜：指文昌、文曲，禄存、天马四曜。
4. 煞曜：指火星、铃星、擎羊、陀罗四曜，又称四煞。有时亦将地空、地劫包括在内，称为六煞。
5. 空劫：指地空、地劫二曜。
6. 化曜：指化禄、化权、化科、化忌四曜。
7. 空曜：指空劫与天空。截空、旬空亦可算作空曜，但力量较弱。
8. 刑曜：指擎羊及天刑。
9. 忌曜：指化忌及陀罗。
10. 桃花诸曜：指红鸾、天喜，咸池、大耗，天姚，沐浴六曜。廉贞贪狼虽有桃花性质，但入正曜系列。
11. 文曜：指化科，文昌、文曲，天才，龙池、凤阁六曜。
12. 科名诸曜：除上述文曜外，加上三台、八座，恩光、天贵，台辅、封诰，天官、天福八曜。
13. 本宫：即主事的宫垣。如看财帛，视财帛宫。财帛宫即是本宫。
14. 对宫：与本宫相对的宫垣。其关系为六冲。如子午冲，子宫与午宫即互成对宫
15. 合宫：与本宫成三合关系的宫垣（隔三宫）。如本宫为子垣，因为申子辰三合，所以申辰两宫垣便是子垣的合宫。
16. 邻宫：相夹本宫的两个宫垣。如本宫在子，则亥与丑两宫便为邻宫。
17. 三方：本宫及合宫，总称为三方。
18. 四正：三方加上对宫，称为四正。
19. 坐守：正曜入躔本宫，称为坐守。如命宫有七杀正曜，称为七杀坐守命宫，或简称七杀守命。
20. 同度：星曜同躔一宫。如除七杀坐守外，宫内又见禄存，则称为禄存同度。
21. 拱照：星曜见于对宫。如七杀坐守，对宫紫微天府，称为紫府拱照。有时又称朝拱。
22. 会照：星曜见于三合宫。如七杀在申宫坐守，贪狼坐辰，破军坐子，故称为贪破会照。
23. 相夹：两星曜位于本宫的邻宫，称为相夹。如寅宫武曲化忌擎羊在卯，陀罗在丑，即为羊陀夹忌。
24. 见：凡星曜会合于三方四正，统称为见。
25. 元冲：凡煞忌诸曜见于三方四正，称为冲，斗数的冲，跟子平的六冲不同。有时亦称为冲破。
26. 垣yuan：即是宫的别名。有时又合称为宫垣。宫垣有两种：一按命盘地支来称谓，如子宫、子垣、丑宫、丑垣之类；一依主事来称谓，如命宫、命垣、父母宫、财帛宫之类。
27. 杀破狼：指七杀、破军、贪狼三曜。这三颗星曜永远在三合宫（即三方）相会，成为命运转变的枢钮，故时合称为杀破狼。
28. 日月：指太阳、太阴二正曜。
29. 入庙：星曜处于最佳状态。有如星曜受人供奉，故称为入庙。（五行被生）
30. 坐旺：星曜状态虽不如入庙时之佳，但却坐临旺宫，星曜依然有力。（五行相同)
31. 落陷：星曜处于最不适宜的环境，以致吉曜无力，凶曜增凶。(五行被克)
32. 平闲：星曜所临的宫垣属于中性。 (五行克生)
33. 借星：凡本宫无正曜，则借对宫星曜入本宫推算，称为借星。无论推算命盘十二宫，或大限十二宫，以至流年、流月、流日、流时，皆需用借星法。

## 认识命盘
不讲排盘，因为现在排盘软件基本做的很成熟了，我直接讲如看懂命盘。
### 整体命盘
[此处需要一张图]
**十二宫**
紫微斗数将命盘划分为12个宫位，分别代表人生不同领域：命、母、福、田、事、友、迁、疾、财、子、妻、兄。
判断各领域的先天吉凶、性格、六亲关系、财运、事业、健康等。

**三方四正**
判断某一宫位的真正吉凶，不能单看本宫，必须看三方四正内的星曜组合、吉煞交集。  

**借星安宫**
当某一宫位内无主星（空宫）时，为判断该宫位吉凶，将对宫的主星借入本宫使用，称为借星安宫。 

借入后，该宫的四化、吉凶以所借之星为主，但仍需参考原宫小星。 例如：命宫无主星，借迁移宫主星，代表外在环境、出外机遇对命主影响极大。

**命盘分类**
- 原局命盘（本命盘）  
判断一个人的先天性格、六亲缘分、适合行业、先天健康强弱。是后续大限、流年吉凶对比的基准盘。
- 大限命盘
将人生分为若干个十年（或数年，依生年干支定），每个十年由某一宫位作为大限命宫，该十年内所有宫位重新定位，形成大限盘。阳男阴女顺行，阴男阳女逆行。
  
判断人生各个十年阶段的主要运势、主题（如青年奋斗、中年成就、老年安养）。需与原局结合：原局为体，大限为用。

**流年命盘**
以某一年的太岁地支为命宫，重新排出的当年运势盘。每年一变，代表该年的吉凶事件、气运起伏。

预测具体年份的财运、感情、健康、事业变动、是非等。需结合大限与原局：原局定本质，大限定十年氛围，流年定具体事件。

河洛派补充重视流年天干四化。
### 单个宫位
- 第一行：主星、辅星、煞星、小星。（代表该宫位内先天带有的主要性格、事件、吉凶倾向）。
- 第二行：庙旺落陷为星曜亮度（决定星曜力量强弱），庙吉最吉陷煞最凶。
- 第三行：四化（禄、权、科、忌）代表该星曜在特定时间（本命、大限、流年）的能量转变。  
禄：顺遂、财富、福气、缘分。
权：掌控、能力、权力、坚持。
科：名声、文采、贵人、化解。
忌：执着、缺失、阻碍、困扰。
- 第四行：流年与小限
流年代表每十年中，此宫位成为流年命宫的年龄。例如10岁、22岁等，该年此宫位主管当年运势。  
小限代表每十二年一轮中，此宫位成为小限命宫的年龄。例如12岁、24岁等，用于更细的年度气运判断。
流年看外，小限看内，流年为主，小限为辅。
- 第五行：神煞与流年结合看运势。
- 第五行：大限代表大限（十年运）的年龄区间，主导10年内的整体运势。
- 第五行：五行局/宫气，三合派会看，比如：沐浴：十二长生之一，主桃花、败地、感情纠葛、新的开始但易破败。
- 第六行：宫名。疾厄宫：主身体健康、疾病、意外、体质强弱，也看心理状态。
- 干支：格局、飞宫四化。
## 十二宫
### 命宫
这是十二宫中最重要的宫位，是整个命盘和斗数的核心，称为"太极点"，是人一生命运及其周围人事物的枢纽所在，在论十二宫的人事物时都要以命宫为基准、为"我"来看。可看出一个人的容貌、性格、才能、思想意识、发展程度、先天运势、机遇的好坏，决定一生的前途，是吉凶成败的关键。在论命时，以命宫为主，三方四正宫为辅，共同决定一生的贵气成就的最高层次，一生命运格局的高低，即先天命数，或称先天运数。先天运势好，后天运势(即大小限)差，遇到困难能得到天助人帮而过关；先天差后天好，则经努力拼搏可适当改造命运，但层次不可能很高，受先天运势的限制。

命宫：生命，本质，性情，人品，才能，发展，人生定位，人生层次。
### 官禄
代表人的功名利禄、职位、工作态度、创业能力、适宜职业、事业发展情况、薪金多寡，以及与师长、上司的关系。可以看其人学生时期的学业情况，若考学时逢事业宫有昌曲魁钺不逢煞破，则发挥良好、考得好。成年人事业宫若逢化忌，则会频繁地更换工作；从政者逢化禄化权，则会升迁而大权在握。还可看配偶对自己的看法和态度。

官禄：工作事业，就业，择业，调动，事业发展与层次，下岗，退休。
### 财帛
看一生经济活动及理财的能力，钱财怎样运用和运用到哪里去，财运的发展情况，收入高低，现金的数量，赚钱能力及赚什么行业的钱财，是正道收入还是横财致富，物质享受是否稳定和充裕。以宫位看求财的方位。

财帛：财运，生意，投资，赚钱，现金，花钱，损财，破财，资金周转，债务。
### 迁移
迁移宫与命宫关系很密切，如影随形，互为表里，命主内，迁主外。代表社会活动的空间、能力、地位，对外发挥的程度、机遇及被社会重用的情形，适应社会环境的能力，在外地活动、出外旅游、远行区交通的吉凶，出外的地点、人际关系及遇到的情况等。与迁动、升迁、调动、搬迁(远地)、离乡、远行等事有关。有无社会上的贵人相助也看此宫。

迁移：活动，出行，外出，外务，出国，向外发展，开职，转业。
### 福德
为思想精神宫，是前世的修为，今生得享受，代表人的寿基、祖荫及先天的福份(包括物质和精神享受)，看是否幸福，是辛劳还是安逸，对物质享受的方式方法，以及得到那一方面的享受；代表个人的修养好坏、生活享受方面的心态、内心世界、思想品德好坏情况；看个人精神是否愉快，是乐观还是悲观，是勤还是懒；代表兴趣、爱好，看爱好是否广泛，根据星情即可以知道其人主要想做些什么，即思想体系；代表财源，来财的地方，看财源多不多，看其人物质生活的优劣、消费方式、钱财的去向、投资环境的好坏和理财能力。

福德：享受，享乐，饮食，娱乐，思想行为，品德，思维活动，财源。
### 田宅
代表家庭、家宅、财库、不动产情况，看家人、家宅是否安宁；看居住环境、布置情況，看祖产的有无及得失情况，能否储钱，置业能力；看家里人对自己的影响；看邻居的好坏。看搬迁(近处)、搬家好坏、库藏情况、不动产买卖的情况，主要看限年田宅宫。

田宅：住房，买卖房子，租房，搬迁，建房，拆房，装修，家具，存钱，财立。
### 父母
代表父母(主要为父亲)的性情、才能、事业、社会背境及吉凶情況；本人与父母的缘份、感情，以及得到父母的恩惠等情況，看是否要靠自己白手成家；也代表长辈、师长、上司，也就是看有无靠山，故就业时，父母宫也可看上司或老板的情况以及与自己的关系是否融合；父母宫又称相貌宫，看人的相貌时以命宫为主，还要参看父母宫；父母宫代表遗传的优步，称为头脑线，看父母对自己的影响，一生的聪明才智以及思想意识、思维方式、学识、文化程度等与此宫有很大关系；父母宫为生我养我益我的宫位，幼年运宜参看父母宫，看少年时生活的好坏以及能否得到父母的关照；父母宫还是文书宫，看与党政文件、法规、文凭、文书、契约、合同、官司等有关的事情。父母宫不好，前半生较为劳心劳力，身心都很累。

父母：遗传，命源，相貌，头脑，意识思维，文化程度，文学修养，才智，名声，前途，靠山，文书，法律，法规，官司。
### 夫妻
代表配偶的容貌、性情、才能、成就情况，看本人的恋爱和婚姻情况，以及夫妻之间的关系、感情、缘份；亦代表本人对配偶的态度、喜爱对象的类型。

夫妻：异性缘，游玩，娱乐，恋爱，婚姻，感情，夫妻、婚后生活事情。
### 子女
代表子女的人数、容貌、性情、好坏、才能、成就发展前景，以及与自己的感情关系，还兆示自己生殖系统情况及能力，生儿育女和夫妻间性生活的情况。

子女：玩乐，游戏，合伙，子女，学生，晚辈，生育，色情，桃花。
### 兄弟
主要代表家庭内兄弟姐妹、母亲的容貌性情才能、成就发展情況，以及兄弟姐妹的多少，相互之间的关系好坏，有无助益。亦可兼看亲密朋友、事业伙伴的关系。

兄弟：财库，成就，费用支出，情感，兄弟情义的事情。
### 奴仆
主要代表与家庭以外一般朋友、平辈、同事、下属、佣人、员工、合作伙伴的关系，看他们是否得力，有无助益；看自己能否服众、人际关系和人缘多少，以及敌人的情况；可以看出"人灾"，若奴仆宫不好，每走到奴仆宫的岁限，便易被小人欺负、骗钱。恋爱对象及婚后的情人也看此宫。

奴仆：交友，人际关系，外交，受骗，恋爱对象，小人损害。
### 疾厄
代表本人的身体，为先天体质、身体状况、病根与健康发展趋势；可以看出人一身最薄弱的部位，灾疾来源，容易发生哪些凶险意外与疾病的种类，以及病伤的部位等。

疾厄：身体，疾病，伤残，死亡，工作地点，工作单位，倒店事。

## 双星系统
### 双星组合的解读逻辑
要理解双星组合，需要把握以下三个核心原则：

**融合型**
同宫的双星没有绝对的主星，而是彼此交融。例如紫微天府同宫，并非紫微为主，天府为辅，而是形成了 帝王+财库 的独特气场，既有紫微的统领欲，又有天府的谨慎守成。

**互补型**
如天机天梁，天机的灵动与天梁的稳重互补，形成善谋略、能决断的特质。

**冲突型**
如廉贞破军，廉贞的细腻与破军的刚烈冲突，形成求变中带着纠结的特质，人生常有颠覆性的选择。

**格局型**：
某些双星组合本身就构成了著名的格局，其寓意远超星曜本身。如天机天梁在辰戌宫同坐命，即为善荫朝纲格，主为人有洞察力、能为人排忧解难。

### 常见双星组合派系
1. 紫微系（紫微+XX）
紫微天府（寅申同宫）
寓意：帝王+财库，格局极大。主稳重、大度，有领导才能和极强的管理能力。但两者都属孤星，内心易有孤傲感，需防因过于保守而错失良机。一生追求事业与财富的稳固。

紫微天相（辰戌同宫）
寓意：帝星+印星，主贵气与权威。为人公正、有原则，重视名誉，适合在体制内或大型机构发展。天相为印，能中和紫微的霸气，使其更有协调力。

紫微七杀（巳亥同宫）
寓意：帝星+将星，主魄力与开创。具有非凡的决断力和执行力，敢于突破常规，人生常有戏剧性变化。七杀的杀气被紫微统御，可将开创力转化为领导力，但过程多艰辛。

紫微破军（丑未同宫）
寓意：帝星+耗星，主颠覆与创新。具有强烈的改造欲，常能打破旧有格局，开创新局。人生起伏较大，事业多变动，适合在变革中求发展。

紫微贪狼（卯酉同宫）
寓意：帝星+桃花主，主物欲与交际。人缘佳，才华横溢，但也易有贪图享受、沉迷欲望的倾向。需防桃花过重影响事业，若在命宫、官禄宫，常有艺术、娱乐、贸易方面的天赋。

2. 府相系组合（天府+XX）
天府天相（命宫、官禄宫或夫妻宫常见）
寓意：府相朝垣格局的基础。主稳重、有助力，一生易得贵人提携，事业稳定。天府为财库，天相为印信，组合在一起有极强的管理能力和协调能力，适合做二把手或核心管理人员。

3. 机月同梁系（天机、太阴、天同、天梁）
天机天梁（辰戌同宫）
寓意：善荫朝纲格。主智慧、洞察力与助人精神。天机的谋略加上天梁的荫护，使此人善于策划、为人排忧解难，适合从事咨询、教育、法律、医疗等智谋+服务型行业。

天机太阴（寅申同宫）
寓意：机月同宫格（与太阴）。主心思细腻、善于策划，对商业、财务、文艺有敏感度。天机的灵动与太阴的柔美结合，使此人具有优雅的气质和极强的分析能力。

天同太阴（子午同宫）
寓意：主福气与享受。性格温和、多才多艺，但需防过于懒散、缺乏进取心。此组合在命宫，一生福泽深厚，但需有煞星或化权激发，方能成大器。

天同天梁（寅申同宫）
寓意：主福寿与荫护。性格豁达、有长者风范，能得人尊敬。天同的福与天梁的寿结合，一生多能逢凶化吉，适合从事公益、教育、医疗等积德之业。

4. 廉杀破狼系（七杀、破军、贪狼）
廉贞七杀（丑未同宫）
寓意：雄宿乾元格。主刚烈、有魄力，能在艰苦环境中脱颖而出。廉贞的次桃花与七杀的肃杀结合，性格刚毅果敢，但需防因冲动而误事。事业上多有突破性成就。

廉贞破军（卯酉同宫）
寓意：主变革与冲突。人生多有颠覆性变化，事业、感情常有波动。廉贞的细腻与破军的刚烈冲突，使内心常有矛盾，需学会在变化中寻找平衡。

廉贞贪狼（巳亥同宫）
寓意：风流彩杖格。主才华与欲望。艺术天赋高、交际能力强，但也易沉迷于酒色财气，需防桃花劫。若命宫有此组合，人生常与娱乐、艺术、社交相关，需有空星（地空、地劫）同宫以去欲，方能转化为清贵之格。

### 总结
双星组合是紫微斗数中极为精妙的语言，它揭示了人性格与命运的复杂性和内在矛盾统一性。它们不是简单的叠加，而是两颗星曜能量的化学反应。

看双星组合，首看融合后的气质，而非单星特质。次看所在宫位，舞台决定气质如何展现。再看辅星与四化，它们是决定组合是成格还是破格的关键。

辅星（尤其是煞星）和四化能极大改变双星组合的走向。
化禄：放大组合的福与财属性，但也可能放大贪欲。
化权：增强组合的魄力与掌控力。
化科：增强组合的名望与专业度。
化忌：凸显组合的缺陷与纠结点。
遇空星（地空、地劫）：对杀破狼和桃花组合有降温和升华作用，能将欲望转化为创意或精神追求。

## 四化
四化是紫微斗数中星曜能量的转化，由天干引发。不同天干会使特定星曜产生禄、权、科、忌的变化，从而改变该星曜的吉凶属性和影响力。
- 生年四化：由出生年天干决定，影响一生。
- 大限四化：由大限宫位天干决定，影响该十年。
- 流年四化：由流年天干决定，影响该年。
- 飞宫四化（河洛派）：以某宫天干起四化，看落入何宫，判断宫位间的吉凶关联。

### 禄
**作用**：
- **增吉**：使星曜的吉性增强，凶性减弱。
- **主顺**：事情顺利、少有阻碍。
- **主财**：财运佳、收入增加、物质丰足。
- **主缘**：人际关系好、贵人运强、桃花（若在桃花星上）。

**看法**：
- 禄入命宫：一生福厚、性格乐观。
- 禄入财帛宫：赚钱容易、财源广进。
- 禄入夫妻宫：感情甜蜜、配偶有助。
- 禄入迁移宫：出外发达、贵人相助。
- **忌见煞星同宫**：禄被冲破，吉力减半。

### 权
**作用**：
- **主掌**：掌握权力、有领导力、能掌控局面。
- **主变**：主动改变、开创、突破。
- **主坚**：意志坚定、不怕困难、能坚持到底。
- **主实**：实质的掌控力，不一定是虚名。

**看法**：
- 权入命宫：性格强势、有领导才能、好胜心强。
- 权入官禄宫：事业掌权、升迁快、适合管理职。
- 权入田宅宫：掌握家中大权、房产置产能力强。
- 权入夫妻宫：对配偶有控制欲，或配偶强势。
- **权忌同宫**：权可制忌，减少忌的负面影响，但过程辛苦。

### 科
**作用**：
- **主名**：名声、声誉、考试顺利、学历提升。
- **主解**：化解冲突、缓和凶象、大事化小。
- **主助**：贵人相助、长辈提携、人缘好。
- **主文**：文采、艺术、学术表现佳。

**看法**：
- 科入命宫：有风度、气质佳、为人谦和、名声好。
- 科入父母宫：父母有教养、学业有师长帮助。
- 科入官禄宫：事业靠名声吃饭、适合学术或艺术工作。
- 科入福德宫：精神生活丰富、有智慧化解烦恼。
- **科可制忌**：科能缓和忌的冲击，使损失减轻。
### 忌
**作用**：
- **主缺**：缺失、不足、遗憾、不圆满。
- **主困**：困扰、阻碍、麻烦、是非。
- **主执**：执着、钻牛角尖、放不下。
- **主耗**：消耗、付出多收获少、身心疲惫。

**看法**：
- 忌入命宫：性格固执、易有烦恼、人生较辛苦。
- 忌入财帛宫：赚钱辛苦、易破财、守财难。
- 忌入夫妻宫：感情执着、易有婚姻问题、付出多回报少。
- 忌入迁移宫：出外不顺、易招惹是非。
- **忌为“所在之处即心之所系”**：忌在哪里，人就执着在哪里，未必全凶，但必有付出。
- **双忌同宫或对宫冲**：凶力加倍，需特别注意。
- **科随忌走**：有科同宫或在对宫可化解部分忌的力量。
###实际判断步骤（三合派）
1. **看生年四化**：定一生格局。
2. **看大限四化**：定十年重点。
3. **看流年四化**：定当年事件。
4. **看四化落入何宫**：判断吉凶作用在哪个领域。
5. **看四化与星曜组合**：例如太阳化忌（父星受损）与太阴化忌（母星受损）含义不同。
6. **看四化与煞星互动**：禄见煞减吉，忌见科减凶。


> 注：四化是紫微斗数中极为重要的动态判断依据，不能孤立看待，必须结合宫位、星曜、格局综合判断。河洛派更强调飞宫四化的因果关系，三合派则重四化与大限流年的配合。

## 原局命盘解法
原局命盘（又称本命盘、天盘）是根据出生年、月、日、时排出的原始命盘，**一生不变**，代表：
- 先天格局与本质
- 性格特质
- 六亲缘分
- 一生运势的大框架
- 是后续大限、流年判断的**基准盘**

### 第一步：定格局
**内容**：判断命盘的总体层次（富贵贫贱、吉凶程度）。
**做法**：
1. **看命宫主星**：单星、双星、空宫。
   - 吉星庙旺：格局较高。
   - 煞星同宫：格局受损。
2. **看三方四正**：命宫、财帛宫、官禄宫、迁移宫的星曜组合。
3. **看特殊格局**：如紫微坐命、日月并明、杀破狼、府相朝垣等。
4. **看生年四化**：禄权科忌的分布。
**用途**：判断命主一生的大致层次。

### 第二步：定十二宫吉凶
**内容**：逐一分析十二个宫位的吉凶，判断各领域的先天运势。
**做法**：
1. **看本宫主星**：星曜种类、亮度（庙旺利陷）。
2. **看辅星吉煞**：左辅、右弼、文昌、文曲、天魁、天钺（吉）；擎羊、陀罗、火星、铃星、地空、地劫（煞）。
3. **看三方四正**：对宫、三合宫的星曜影响。
4. **看本宫四化**：是否有生年四化落入。

**十二宫对应领域**：
| 宫位 | 主看 |
|------|------|
| 命宫 | 性格、格局、整体运势 |
| 兄弟宫 | 兄弟姐妹缘分、朋友助力 |
| 夫妻宫 | 婚姻、感情、配偶特质 |
| 子女宫 | 子女缘分、生育、下属 |
| 财帛宫 | 财运、赚钱方式、理财能力 |
| 疾厄宫 | 健康、疾病、体质 |
| 迁移宫 | 外出、人际、机遇、外在形象 |
| 交友宫 | 朋友、同事、合作伙伴 |
| 官禄宫 | 事业、工作、适合行业 |
| 田宅宫 | 房产、家庭、晚年运 |
| 福德宫 | 精神生活、享福、内心世界 |
| 父母宫 | 父母缘分、长辈、文书 |

### 第三步：看四化分布
**内容**：分析生年四化（禄、权、科、忌）落入何宫，判断一生重点。

**做法**：
- **化禄**：福气、财运所在之处。
- **化权**：能力、掌控所在之处。
- **化科**：名声、贵人所在之处。
- **化忌**：执着、缺失所在之处。

**示例**：
- 化禄入财帛宫 → 一生财运佳。
- 化忌入夫妻宫 → 感情易有执着与波折。

### 第四步：看三方四正
**内容**：判断某一宫位的真实吉凶，不能单看本宫。
**做法**：
- 以命宫为例，必须同时看：
  - **命宫**（本宫）
  - **财帛宫**（命宫的三合）
  - **官禄宫**（命宫的三合）
  - **迁移宫**（命宫的对宫）
**原则**：
- 三方四正吉星多 → 该领域顺遂。
- 三方四正煞星多 → 该领域波折。
- 吉煞交集 → 吉中藏凶或凶中有救。

### 第五步：看特殊格局

**内容**：判断命盘是否形成特殊格局，格局决定格局层次。

**常见格局（三合派）**：
| 格局 | 组成 | 寓意 |
|------|------|------|
| 紫微坐命 | 紫微在命宫 | 帝王格，有领导力 |
| 杀破狼 | 七杀、破军、贪狼在命迁线 | 变动、开创、大起大落 |
| 府相朝垣 | 天府、天相在三方 | 稳重、富贵 |
| 日月并明 | 太阳、太阴庙旺 | 光明磊落、福厚 |
| 机月同梁 | 天机、太阴、天同、天梁 | 适合公职、策划 |
| 巨机同临 | 巨门、天机同宫 | 口才、是非并存 |

### 第六步：看空宫与借星
**内容**：处理无主星的宫位。
**做法（三合派）**：
- 某宫无主星 → 借对宫主星使用。
- 借星后，该宫的吉凶以所借之星为主。
- 例如：命宫空宫，借迁移宫主星 → 命主受外出、环境影响大。


### 第七步：看五行局与命主、身主
**内容**：辅助判断性格与运势节奏。
**做法**：
- **五行局**：水二局、木三局、金四局、土五局、火六局 → 决定大限起运年龄与运势节奏。
- **命主**：命宫所属的命主星 → 辅助判断性格核心。
- **身主**：生年支所属的身主星 → 辅助判断后天运势。

### 原局命盘解法的总结流程
1. 定格局 → 判断一生层次
2. 定十二宫 → 判断各领域先天吉凶
3. 看四化 → 判断一生重点与执着
4. 看三方四正 → 综合判断宫位真实吉凶
5. 看特殊格局 → 提升或降低格局层次
6. 看空宫借星 → 处理无主星宫位
7. 看五行局、命主、身主 → 辅助判断

## 大限盘解法
| 项目 | 大限 |
|------|------|
| 周期 | 十年 |
| 起法 | 以命宫起，阳男阴女顺行，阴男阳女逆行 |
| 代表 | 十年阶段的主要运势、人生主题 |
| 力量 | 强，主导十年氛围 |
| 与原局关系 | 原局为体，大限为用 |

**核心原则**：
- **原局定本质**（先天格局）
- **大限定氛围**（十年主题）

### 第一步：看大限命宫

**做法**：
- 该十年内，以该宫位为“大限命宫”。
- 看该宫的原局星曜、亮度、吉煞、四化。

**用途**：
- 判断该十年的总体吉凶基调。
- 例如：大限命宫吉星庙旺 → 十年顺遂；凶星煞星多 → 十年波折。

### 第二步：看大限四化

**做法**：
- 以大限宫位的天干起四化（禄、权、科、忌）。
- 看四化飞入何宫，判断该十年的重点事件领域。

**示例**：
- 大限宫天干为甲（甲廉破武阳）：
  - 化禄入财帛宫 → 十年财运佳。
  - 化忌入夫妻宫 → 十年感情易出问题。

**用途**：
- 大限四化是十年运势的**动态核心**，比原局四化更具体。

### 第三步：看大限与原局的互动
**做法**：
- 大限命宫叠原局某宫 → 该原局领域的十年应期。
- 大限四化入原局某宫 → 该原局领域被引动。

**原则**：
- 原局吉星 + 大限吉化 → 大吉。
- 原局凶星 + 大限忌化 → 大凶。
- 原局吉星 + 大限忌化 → 吉中藏凶。
- 原局凶星 + 大限吉化 → 凶中有救。

### 第四步：看大限三方四正
**做法**：
- 以大限命宫为本宫，看其三方四正（财帛、官禄、迁移）的星曜组合。
**用途**：
- 综合判断该十年的吉凶程度，避免单宫误判。

### 大限看法的总结要点
| 重点 | 做法 |
|------|------|
| 大限命宫 | 定十年基调 |
| 大限四化 | 定十年重点事件领域 |
| 原局叠宫 | 定哪些原局领域被引动 |
| 三方四正 | 综合判断吉凶 |


## 流年解法
| 项目 | 流年 |
|------|------|
| 周期 | 一年 |
| 起法 | 以当年太岁地支为命宫 |
| 代表 | 该年的具体事件、吉凶应期 |
| 力量 | 较弱于大限，但决定具体应事 |
| 与原局、大限关系 | 原局为体，大限为纲，流年为目 |

**核心原则**：
- **原局定本质**（先天格局）
- **大限定氛围**（十年主题）
- **流年定应事**（具体事件）

### 第一步：看流年命宫
**做法**：
- 看流年命宫的原局星曜、亮度、吉煞。
- 看流年命宫的三方四正。

**用途**：
- 判断该年的总体吉凶。

### 第二步：看流年四化

**做法**：
- 以流年天干起四化（如丙年：丙同机昌廉）。
- 看流年四化飞入何宫，判断该年的具体事件。

**示例**（2026丙午年，丙同机昌廉）：
- 天同化禄 → 福气、享受增强。
- 天机化权 → 变动、策划能力增强。
- 文昌化科 → 考试、名声有利。
- 廉贞化忌 → 感情、是非、血光要注意。

**用途**：
- 流年四化是年度运势的**核心驱动力**。

### 第三步：看流年与大限、原局的互动

**做法**：
- 流年命宫叠大限命宫 → 该年是大限十年中的关键年。
- 流年四化入大限四化宫位 → 该年应大限的重点事件。
- 流年四化入原局吉凶宫 → 原局吉凶在该年应事。

**原则**：
- 原局为根，大限为干，流年为枝叶。
- 三者吉凶一致 → 应事明显。
- 三者矛盾 → 吉凶减轻或反复。

### 第四步：看流年神煞

**做法**：
- 看当年太岁地支对应的流年神煞（如白虎、丧门、天德、月德等）。

**常见流年神煞**：

| 神煞 | 寓意 |
|------|------|
| 天德、月德 | 贵人、逢凶化吉 |
| 白虎 | 血光、意外、是非 |
| 丧门、吊客 | 丧事、病痛 |
| 太阳 | 男性贵人、光明 |
| 太阴 | 女性贵人、暗财 |

**用途**：
- 辅助判断该年的特殊事件（如血光、贵人、丧事）。


### 第五步：看小限（辅助）
**做法**：
- 小限反映该年的主观心态、自我选择。
- 流年主客观事件，小限主主观心态。

**用途**：
- 流年吉 + 小限吉 → 顺利且心态好。
- 流年吉 + 小限凶 → 有机会但自己放弃。
- 流年凶 + 小限吉 → 有波折但能平稳度过。


## 四、流年看法的总结要点

| 重点 | 做法 |
|------|------|
| 流年命宫 | 定年度基调 |
| 流年四化 | 定年度具体事件 |
| 叠大限、原局 | 判断应期与深度 |
| 流年神煞 | 辅助判断特殊事件 |
| 小限 | 辅助判断主观心态 |
